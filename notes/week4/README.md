# Week 4 Examples

- [address_demo.cpp](address_demo.cpp): demonstrating the unpredictable nature
  of pointer addresses
- [dangling_pointer_demo.cpp](dangling_pointer_demo.cpp): examples of dangling
  pointers
- [memory_leak_demo.cpp](memory_leak_demo.cpp): example of a memory leak
- [double_deletion_demo.cpp](double_deletion_demo.cpp): examples of deleting the
  same memory twice
- [swap_demo.cpp](swap_demo.cpp): examples of swapping variable values using
  pass by value, pass by reference, and pass by pointer
- [c_string_demo.cpp](c_string_demo.cpp): demonstrates working with C-strings
- [c_string_funcs.cpp](c_string_funcs.cpp): simple implementations of the
  C-string functions `strlen`, `strcpy`, and `strcat`
- [cpp_string_demo.cpp](cpp_string_demo.cpp): demonstrates working with C++
  strings
- [stringBuffer.cpp](stringBuffer.cpp): implementation of a simple string buffer
  that improves on the C-string functions (but not as easy as `std::string`)
